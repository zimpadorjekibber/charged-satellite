# Table Management & Printing Workflow

## Table Management (Admin)

1.  **Access Admin Dashboard**: Log in as an admin.
2.  **Add Table**:
    *   Navigate to the "Table Management" section.
    *   Enter a table name (e.g., "Patio 1") in the "Add New Table" form.
    *   Click "Add Table". The table will appear in the list.
3.  **Edit Table**:
    *   Find the table in the list.
    *   Click "Edit".
    *   Enter the new name in the prompt.
4.  **Delete Table**:
    *   Click "Delete" on the table card. Confirm the action.
5.  **QR Codes**:
    *   QR codes are automatically generated for all tables.
    *   Click "Print QR Card" to open a printable view of the QR code for a specific table.
